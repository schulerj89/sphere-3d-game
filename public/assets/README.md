# Assets

Bundled third-party media is catalogued in [asset-manifest.json](./asset-manifest.json), with license and provenance in [docs/ASSET_SOURCES.md](../../docs/ASSET_SOURCES.md) and [docs/ATTRIBUTIONS.md](../../docs/ATTRIBUTIONS.md).

Add an asset only when its source URL, exact license, intended use, byte size, and fallback behavior are recorded first.
